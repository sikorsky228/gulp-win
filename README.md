# Gulp for Windows with SASS

name of packages:
* gulp-sass
* gulp-cssmin
* gulp-rename
* gulp-autoprefixer
* gulp-uncss
* browser-sync
* gulp-uglify
* gulp.spritesmith


###tasks

default task:
> gulp

compile sass, minify css, autoprefix, minify js, add local server and watch with the changes in sass,js,html

Delete unused css:
> gulp uncss

Build sprites

> gulp sprite
