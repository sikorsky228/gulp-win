# Gulp for Windows with SASS

name of packages:
* gulp-sass
* gulp-cssmin
* gulp-rename
* gulp-autoprefixer
* gulp-uncss
* browser-sync
* gulp-uglify
* gulp.spritesmith
* gulp.imagemin
* 

###tasks

default task:
> gulp

compile sass, minify img, css, autoprefix, minify js, add local server and watch with the changes in sass,js,html,img files

Delete unused css:
> gulp uncss

Build sprites

> gulp sprite
